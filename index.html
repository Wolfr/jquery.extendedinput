jquery.extendedinput.js

Show extra HTML elements based on changes in form elements. Useful in HTML prototyping.

Demo

Radio and checkboxes

This pattern shows HTML elements based on selection of a radio button or check box.

The "link" is made through a data attribute (data-show) and the ID of the radio button/checkbox.

Note: you have to apply the initial state manually (add your hide classes to the elements that should be hidden).

  • Run npm install and bower install

To build

  • Make sure you have grunt installed
  • Run grunt
