jquery.extendedinput.js

Show, hide or toggle HTML elements based on click and change Javascript events. Useful in HTML prototyping.

Using links

The simplest use case is to use a link to trigger a change.

Changelog

2.0.0
  • This release cleans up the functionality provided in jquery.extendedinput 1.x. The main focus is on making the plugin simpler to use. The syntax to show, hide and toggle was simplified. The only attributes you will ever need are data-show,data-hide and data-toggle.
1.1.0
  • Unified data toggle (#5)
  • Updated demo markup to use Bootstrap 3
1.0.0

Contributing

To run locally

  • Make sure you have node js and bower installed
  • Git clone the repository git clone git@github.com:Wolfr/jquery.extendedinput.js
  • Run npm install and bower install

To build

  • Make sure you have grunt installed
  • Run grunt
